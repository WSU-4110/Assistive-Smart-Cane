# Assistive Smart Cane

**Folders**
- /app — web app (Vite/TypeScript)
- /firmware — Arduino code + pinout
- /hardware — wiring diagram
- /docs — SRS, UML, sprint notes

**Run web app**
```bash
cd app
npm install
npm run dev
```
**Upload firmware**
- Open `/firmware/AssistiveSmartCane.ino` in Arduino IDE
- Board: Arduino Uno
- Upload
